# 42 Piscine PHP

### The Piscine PHP is a 2 weeks intensive program to learn the basics of PHP at the 42 coding school. Exercices are grouped into "days", and each day has to be turned in within a short timeframe; and during the two weekends, a bigger group project, called a "rush", has to be completed.

| [02](#day02) | 05-22 | 05-23 | RegEx, Files, cURL |
| [03](#day03) | 05-23 | 05-24 | PHP with a server (MAMP), Superglobals |
| [04](#day04) | 05-24 | 05-25 | User Sessions, Hashing |

| [09](#day09) | 05-31 | 06-01 | JavaScript, jQuery, AJAX |
